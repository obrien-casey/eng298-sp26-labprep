## Markdown Basics
Markdown is a lightweight formatting language used on GitHub for README files, documentation, and more. You can learn more about basic syntax here:  

👉 [**Markdown Guide – Basic Syntax**](https://www.markdownguide.org/basic-syntax/)

Try adding a few simple Markdown elements:
- `#` for headings  
- `**bold**` or `*italic*` for emphasis  
- `-` for bullet lists  
- `` `code` `` for inline code
